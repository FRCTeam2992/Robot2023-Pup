# Controller B Guitar to XBox

## Neck buttons

- Green = XBox A
- Red = XBox B
- Yellow = XBox Y
- Blue = XBox X
- Orange = XBox Left Bumper

## Body

- Minus = XBox Back/Select
- Plus = XBox Start
- Strummer = XBox reverse direction Dpad vertical axis
- Joystick = XBox left stick (no push button)
- Whammy Bar = XBox left trigger
